# Test Data Directory

This directory contains test data files for the edi-validation integration tests.

## Files

- `test_orders.edi` - Sample EANCOM ORDERS document
- `test_desadv.edi` - Sample DESADV document
- `test_invoic.edi` - Sample INVOIC document
- `invalid_syntax.edi` - Document with syntax errors
- `missing_required.edi` - Document missing required segments
- `invalid_codes.edi` - Document with invalid code list values

## Notes

These are placeholder files for the integration tests. The actual EDI content would be generated or loaded from external sources in a real implementation.
